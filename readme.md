# UAV-ISAC-Pub — 资料汇总（站点骨架）

本仓库是关于 UAV 与 ISAC（Integrated Sensing and Communications）方向的论文与资源汇总，灵感来源于 yuanhao-cui/Must-Reading-on-ISAC。

文件结构
- README.md — 仓库说明
- docs/reviews.md — 综述与教程（占位，暂不包含文章）
- docs/papers.md — 论文索引（占位，暂不包含文章）
- docs/code.md — 开源代码与实现（占位）
- docs/datasets.md — 数据集（占位）

条目格式
- YYYY — 标题 — 作者 — 链接(ArXiv / DOI) — [关键词/标签]

如何贡献（推荐流程）
1. Fork 本仓库。
2. 在对应 docs/*.md 或 data/ 文件中添加条目（请勿直接复制其他仓库的整段内容，若引用请注明来源并遵守原许可）。
3. 发起 Pull Request，说明新增内容来源与分类。
